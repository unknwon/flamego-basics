# 《Flamego 框架基础》

《Flamego 框架基础》是一套针对 Go 语言 Web 框架 [Flamego](https://github.com/flamego/flamego) 出品的视频语音教程，主要面向 Web 应用开发者。

## 基本信息

- **教程讲师**：[无闻](https://github.com/unknwon)
- **语言版本**：Go 语言 >= 1.16
- **框架版本**：Flamego >= 1.2.0
- **开发工具**：[Visual Studio Code](https://code.visualstudio.com/)
